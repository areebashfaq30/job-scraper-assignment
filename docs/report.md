# Job Market Analysis Report

## Sources
3 public Greenhouse careers pages:
- Uber: https://boards.greenhouse.io/uber
- Shopify: https://boards.greenhouse.io/shopify
- DoorDash: https://boards.greenhouse.io/doordash

Search: "software engineer", filter: Remote/US.

Collected ~50-100 listings (depends on run).

## Key Findings (from analysis/analyze.py)
- **Top Skills**: Python, JavaScript, AWS, React, SQL, Docker (tech-heavy for software roles).
- **Top Locations**: Remote (60%+), San Francisco, New York, Toronto.
- **Top Companies**: Uber (most postings), Shopify, DoorDash.
- **Entry-level**: ~15% (intern/junior roles).
- **Common Titles**: Software Engineer, Senior Software Engineer, Backend Engineer.
- **Employment**: Mostly Full-time.

## Insights
High demand for cloud/full-stack skills. Remote work dominant post-COVID. Uber hiring aggressively for engineering.
